# AppScreen

AppScreen is a desktop application manager, using the
Kivy graphical user interface (GUI). Enabling agile
and effective creation and development

version 0.6.1-dev

## How to use?

See simple code code:
```
from app.layout import Blank

app = Blank(
    app_name='My App'
    icon='myicon'
)

app.run()
```
